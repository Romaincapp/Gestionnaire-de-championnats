# Feature

Implémenter une nouvelle fonctionnalité.

## Étapes

1. **Clarifier**
   - Objectif de la feature ?
   - Utilisateurs concernés ?
   - Critères d'acceptation ?
   - Contraintes techniques ?

2. **Explorer**
   - Où s'intègre la feature ?
   - Code similaire à réutiliser ?
   - Patterns du projet ?

3. **Concevoir**
   - Fichiers à créer/modifier
   - Interfaces/types nécessaires
   - Dépendances à ajouter
   - **Proposer le plan AVANT de coder**

4. **Implémenter**
   - Commencer par le cœur de la logique
   - Suivre les conventions existantes
   - Gérer les cas d'erreur

5. **Intégrer**
   - Connecter avec l'app
   - Routes/endpoints si nécessaire
   - Navigation/UI

6. **Tester**
   - Tests unitaires
   - Tests manuels
   - Cas limites

7. **Finaliser**
   - Nettoyer (logs, TODOs)
   - Build OK
   - Préparer commit
